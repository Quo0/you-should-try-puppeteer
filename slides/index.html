Что такое Puppeteer ?

Библиотека, которая предоставляет удобное API для работы с Chrome Devtools Protocol

CDP

СDP может все!

Puppeteer может не все, но многое!

Зато он позволяет абстрагироваться
и писать более семантичный код

А если чего-то действительно нехватает, всегда есть возможность подключитсься к CDP сессии и запустить это вручную

class Puppeteer

Ответствен за установку соединения с браузером

.launch()

- запускаем и конектимся к браузеру скаченному в node_modules

- либо с флагом executablePath указываем путь до другого браузера

.connect()

- конектимся к уже запущеному браузеру указав ссылку на WebSocket


Оба метода возвращают экземпляр Браузера

class Browser, class BrowserContext

Browser

.createIncognitoBrowserContext()

- создаем контекст с изолированной сессией

.browserContexts()

- получаем все контексты

.pages()

- получаем все страницы всех контекстов

BrowserContext

.newPage()

- создаем страницу в текущем контексте

.pages()

- получаем все страницы текущего контекста

class Page
class Frame
class ExecutionContext

Page

- служит агригатором методов для работы с конкретной табой / экстэншеном

- имеет как минимум один frame - mainFrame

- имеет доступ к Worker'ам

Frame

- содержит основные методы для работы

- имеет как минимум один контекст исполнения

ExecutionContext

- служит для работы с доменом Runtime
